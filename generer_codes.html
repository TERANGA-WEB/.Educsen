Configuration EmailJS (à faire une fois)

  1. Créer un compte sur emailjs.com (gratuit jusqu'à 200 emails/mois)
  2. Créer un Service (ex: Gmail) → noter le Service ID
  3. Créer un Template avec les variables : from_name, from_email, ecole, plan, montant, reference, message, date
  4. Dans abonnement.html, remplacer EMAILJS_SERVICE_ID, EMAILJS_TEMPLATE_ID, EMAILJS_PUBLIC_KEY
  5. Dans subscription.js, ajouter l'import dans chaque page protégée (voir commentaire en haut du fichier)
